# Xenia

## Описание
Этот проект разрабатывается в рамках магистерской диссертации на тему  
**"Алгоритмические подходы к расчетам отражений в реальном времени для игр и интерактивных приложений"**.  
Цель работы — исследование и реализация методов оптимизированного расчета отражений с использованием современных графических API и алгоритмов.

## Технологии
- **Язык:** C++23
- **Сборка:** CMake + VCPKG
- **Графика:** Vulkan
- **Среда разработки:** Visual Studio 2022
